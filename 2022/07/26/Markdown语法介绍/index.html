Markdown语法介绍

Title：Markdown语法介绍

1.大纲级别介绍

一级大纲 ：一个#加上一个空格

二级大纲 ：两个#加上一个空格

三级大纲 ：三个#加上一个空格

四级大纲 ：四个#加上一个空格

五级大纲 ：五个#加上一个空格
六级大纲 ：六个#加上一个空格

2.分割线

至少三个减号

3.字体

斜体文本 ：两个星号之间的文章段落为斜体
粗体文本 ：两个星号和两个星号之间的文章段落为粗体
粗斜体文本 ：三个星号和三个星号之间的文章段落为粗斜体

4.删除符号

今天是晴天
两个波浪线和两个波浪线之间（波浪线在键盘上位于数字1左边）

5.下划线

今天是晴天
在要下划线的文字左边加上尖括号，里头加上小写字母u；在文字右边也加上尖括号，里面加上/u

6.列表

无序列表

  • 第一项
  • 第二项
  • 第三项
    在第一项前面加上一个“-”和一个空格，转行时第二项自动出现。结束列表，不按回车键进入下一行

有序列表

  1. 第一项
  2. 第二项
  3. 第三项
    在第一项前面加上1.和一个空格，转行时第二项自动出现。结束列表，不按回车键进入下一行

列表的嵌套

  1. 第一大项：
    • 第一项
    • 第二项
    • 第三项
  2. 第二大项：
    • 第一项
    • 第二项
    • 第三项

7.引用

引用
一个大于号加上一个空格

8.代码

print f( )
两个反引号之间加上函数（反引号在键盘上位于数字1左边）

9.代码段

三个反引号+代码段落+三个反引号，两组反引号在两行

10.markdown超链接

[百度] (https://www.baidu.com)
中括号（英文）里加上网站名，加上小括号（英文）里加上网站链接

11.markdown表格

: —– :—-: —- :
向左对齐 居中 向右对齐

12.上传图片

  1. 上传图片到[路过图床] (https://imgchr.com/)
  2. 复制Markdown code
